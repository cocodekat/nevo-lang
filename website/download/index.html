  • Includes Nevo and TCC both required for compiling

Installation instructions

Windows setup
  1. Unzip the installed folder.
  2. Then run install.bat this will install nevo to your computer.
  3. Verify installation:
  4. C:\nevo\nevo.exe
  5. This should print something about how to use the program.
macOS setup
  1. Unzip the installed folder.
  2. Then run install.sh this will install nevo to your computer.
  3. Verify installation:
  4. $HOME/nevo/nevo
  5. This should print something about how to use the program.
